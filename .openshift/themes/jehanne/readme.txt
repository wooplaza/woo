## About Jehanne

Description: Jehanne is a responsive theme with custom options. It has two predefined color schemes (light and dark) and custom colors, slider, layout options, CSS3 and JQuery animation and more. You can make a unique design by using different widgets, sidebars and custom colors. The complete website can be customized using WordPress default Theme Customizer by previewing your changes in real time. Supports WooCommerce plugin. Translation ready. You can find more info and demo at http://wpblogs.ru/themes/blog/theme/jehanne/.

## How to install

* Unzip and then copy the jehanne directory to /wp-content/themes/ in your WordPress installation. 
* Activate from admin menu. 

## How to use a theme

* Theme settings placed in the WordPress Theme Customizer at "Appearance >> Customize".

Jehanne has options for:

Sticky settings: you can make header section or second top menu 'Sticky'.

Logotype: Image at the left top corner of your website. This Image show up at max-width 300px. Should be an Image with size 300px x [50px-100px] or so.

Site title & Tagline: use it for Site Title & Tagline. Can be empty.

Layout: You can set here:

	Max Width of the site (number between 960 and 1349: width in pixels)
	Margin Top in pixels
	Margin Bottom in pixels.

Color Scheme: You can pick theme colors in this section:

Dark and light colors for content area (Color Scheme) and Blue, Black, Red color skin for sidebars and menus. Blue color skin works better with light color scheme and red/black with dark color shceme. This option refresh theme colors (reset for colors). Colors can be changed in the next section.

Colors

This section has 32 colors for elemets (link, h1-h6, header text, menus and sidebars) and optional gradient for menus. Gradient uses 2 colors: first gradient color and menu color. Gradient has an opacity option too. Use gradient style "none" to switch off this option.

Header Image

Section for header image. 300px x 300px.

Front Page style

Style of widgets at page with front page template.

Featured Image

Position of Featured image: Left (40%), Right(40%) and Small (right, has no margins, width 300px).

Background Image

Background Image for the website.

Navigation

4 menus and 4 checkboxes to hide/show menus. Refresh Customizer to apply hide/show option.

Mobile

Options for smaller screens:

	Responsive or not
	Min with for not Responsive
	Hide or not top menus/sidebars behind a button on mobile	This section can be untouched.

Social Media Links

You can choose:

	display Links or not (above the header Image)
	size (small/big)
	20 Links

You can use links from this section in the Jehanne Social Icons widget too (Appearance >> Widgets +++ Drag-and_drop into sidebar).

Scroll to top button

In this section you can choose:

	display or not this button
	position
	jump or not.

Favicon 

Should be icon Image [16x16]px.

Static Front Page

Display Blog (your latest posts) or some page on first page of your website. You can choose 'A static page' and set Front Page Template for it (at page's Edit screen) to have different sidebars on your front page (different column/header/footer).

Demo Widgets

Jehanne has 4 demo widgets in the second footer sidebar for demonstration purpose only. You can replace widgets by dragging-and-dropping your own widgets into footer sidebar. You can switch off widgets in the footer at Appearance >> Customize >> Demo Widgets. This Demo section uses Jehanne Image widget and Standard Text Widget. You can add widgets to the header area or the footer area with your own content.

Sample HTML for styling header/footer via Text Widget:

<p style="
		border-bottom: 1px dashed #fff; 
		display: block; 
		margin-bottom: 0; 
		text-align:center; 
		font-size: 74px; 
		background-color: #000; 
">
	Some Styled text here.
</p>
<p style=" 
		display: block;
		margin-bottom: 0;
		background-color: #000; 
		text-align:center; 
		color: #1e73be;
		padding-bottom: 5px)
">
	Some Styled Description here.
</p>

* Theme widgets can be found at "Appearance >> Widgets" page, 

Jehanne has many sidebars. It has different sidebars for posts, pages and the front page template. 

Structure of the Front Page Template can be found here:

http://wpblogs.ru/themes/jehanne-structure/

Jehanne has 3 custom widgets: 

	slider/news
	Jehanne Image with css3-JQuery animation effects
	Social Icons Widget

Instruction on how to configure Slider can be found at http://wpblogs.ru/themes/blog/configure-sliders-for-your-website/

* Page templates

Jehanne has front page template with different sidebars (8 sidebars: 3 top sidebars, content sidebar, 3 footer sidebars and 1 column). You can use only one sidebar or all sidebars at your website.

Jehanne has full width/no content page template without column.
